MILESTONE 0 · NGINX ALONE

It works. 🎉

You're looking at a static file served by nginx, running inside a Docker container, reached through the port mapping 8080 → 80.

What just happened, in order:

  • Your browser asked localhost:8080 for /
  • Docker forwarded it to port 80 in the container
  • nginx matched location / and served index.html
